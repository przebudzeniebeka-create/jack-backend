# Jack Backend (Flask)

## Lokalne uruchomienie
python -m venv .venv
# Windows:
.venv\Scripts\activate
# macOS/Linux:
# source .venv/bin/activate

pip install -r requirements.txt
cp .env.example .env   # uzupełnij klucze (lub ustaw ręcznie w systemie)
python app.py          # http://127.0.0.1:5000/api/health

## Deploy na Railway (free)
1) GitHub → wrzuć repo
2) Railway → New Project → Empty Project → Connect to GitHub
3) Variables:
   - OPENAI_API_KEY=...
   - (opcjonalnie) ELEVENLABS_API_KEY=..., ELEVENLABS_VOICE_ID=...
   - (opcjonalnie) CORS_ORIGIN=https://jackqs.ai,https://www.jackqs.ai
   - (na start bez DATABASE_URL => użyje SQLite)
4) Deploy → sprawdź:
   https://<twoj-projekt>.up.railway.app/api/health

## DNS (Cloudflare)
CNAME:
  Name: api
  Target: <twoj-projekt>.up.railway.app
  Proxy: ON (pomarańczowa chmurka)

Test:
curl -s https://api.jackqs.ai/api/health
